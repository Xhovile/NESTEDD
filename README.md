# NESTÉDD

**NESTÉDD** is a Malawi-focused accommodation and property platform designed to make housing search, listing management, and property discovery more structured, trustworthy, and scalable.

The platform is being built to support:

- Houses for rent
- Houses for sale
- Hostels
- Lodges
- Hotels
- Property-related service add-ons

It is designed around a mobile-first experience and a verified-listings model to reduce confusion, improve trust, and create a more professional property marketplace.

---

## Vision

To build a credible digital housing infrastructure for Malawi where tenants, students, landlords, agents, and property operators can interact through a more transparent and organized platform.

---

## Core Product Direction

NESTÉDD is planned as more than a listings site. It is being structured as a full property ecosystem with:

- smart search and filtering
- verified listing workflows
- listing status management
- agent and admin dashboards
- hostel discovery by institution
- sale lead capture and routing
- cleaning and photography service add-ons
- future-ready payment and messaging capabilities

---

## Listing Status Logic

NESTÉDD is being designed around clear property status control:

- **VACANT** → visible
- **OCCUPIED** → hidden
- **FULL** → hidden
- **SOLD** → hidden

This keeps live results relevant while preserving internal data for future relisting, analytics, and operational control.
